# 今日酒价微信公众号链接采集说明

这个目录保存从微信公众号「今日酒价」里采集到的「批发参考价」相关文章链接，供后续小龙虾或 AI 批量抓取、转 Markdown、抽取价格数据使用。

目录名不绑定年份。新增年份时，继续按 `YEAR-links.csv`、`YEAR-missing-dates.csv`、`YEAR-summary.json` 的方式追加文件即可。

## 当前文件

- `2025-summary.json`: 2025 年机器可读摘要，包括总数、缺失数、月度覆盖数和特殊复查结论。

## 采集和校验方法

采集是通过微信桌面客户端手动/自动操作完成的，不是通过搜狗或全网搜索。

可靠路径如下：

1. 打开微信公众号「今日酒价」。
2. 点击公众号窗口内的搜索按钮。
3. 输入目标查询，例如 `2025年12月31日 批发参考价`，或按实际微信搜索习惯输入日期加关键词。
4. 在搜索结果里只使用 `今日酒价 发表的文章` 这一组结果。
5. 如果需要进入更多结果，只点击这一组右侧的 `更多`。
6. 打开候选文章后，通过文章右上角菜单复制链接。
7. 抓取复制到的 `mp.weixin.qq.com` 页面，读取微信页面里的发布时间。
8. 只有当发布时间等于目标日期时，才把 URL 写入对应年份的 `YEAR-links.csv`。

重要：不要使用微信搜索结果顶部的全局 `文章` 标签页。那个标签页会搜索所有公众号文章，不限于「今日酒价」，会混入其他账号内容。

## 后续处理建议

后续批处理可以直接读取对应年份的链接文件：

```text
date,url
2025-12-31,https://mp.weixin.qq.com/s/...
```

下载和 Markdown 转换时，图片保留微信远程链接即可，不需要下载到本仓库。

如果后续再次补采，建议继续沿用同一条校验规则：只有「公众号账号内结果」且「微信文章发布时间等于目标日期」的 URL 才能进入链接文件。
